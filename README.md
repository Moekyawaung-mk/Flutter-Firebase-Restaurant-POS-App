# Restaurant POS MM

Flutter + Firebase restaurant POS mobile app with Burmese language support.

## Features

- Restaurant code login
- Role based access
  - Admin
  - Waiter
  - Kitchen
  - Cashier
- Table management
- Menu ordering
- Real-time kitchen updates
- Billing with discount and tax
- Payment records
- Daily sales report
- PDF receipt printing
- Burmese / English language toggle

## Tech

- Flutter
- Firebase Auth
- Cloud Firestore
- Provider
- PDF / Printing
